Modelo de caixas

O que é uma caixa?

Segundo o conceito "box model" a grande maioria dos elementos HTML são como caixas. Elas são containers que armazenam conteúdos ou até mesmo outras caixas.

Anatomia de uma caixa

Principais elementos da anatomia de uma caixa 

        content = conteúdo
        height = altura
        width  = largura
        padding = preenchimento, borda de dentro
        border = borda
        outline = contorno
        margin = margem, borda de fora

Tipos de caixa

Box-level

Características

  • Sempre inicia numa linha nova
  • Sempre ocupa a largura total da tela
  • Novos elementos começam na próxima linha

    • Exemplos de box-level

  • <div>
  • <h1> - <h6>
  • <p>
  • <main>
  • <header>
  • <nav>
  • <article>
  • <aside>
  • <footer>
  • <form>
  • <video>

    • Inline-level

    Características

  • Continua na mesma linha
  • Ocupa só o necessário
  • Não pula pra próxima linha

    • Exemplos de inline-level

  • <span>
  • <a>
  • <code>
  • <small>
  • <strong>
  • <em>
  • <sup> - <sub>
  • <label>
  • <button>
  • <input>
  • <select>
