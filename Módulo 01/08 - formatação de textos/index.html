Principais formatações

Negrito / Destaque

Nesta frase temos um termo em destaque usando a tag <b> (não semântica, apenas visual).

Nesta frase temos um termo em destaque usando a tag <strong> (semântica, pois há significado, dizemos que a frase é de alguma importância)

Itálico / Ênfase

Nesta frase temos um termo em ênfase usando a tag <i> (não semântica)

Nesta frase temos um termo em ênfase usando a tag <em> (semântica).

Texto marcado

Podemos crias também um texto marcado usando a tag <mark>

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno, usando as tags <big> (obsoleto, pois só da forma) e <small> (o prof acha que não está obsoleto pq existe as "letras pequenas de contratos e coisas assim, enão teria algum significado)

Texto deletado

Podemos marcar o texto como um texto excluído para indicar que el deve ser lido, mas não considerado.

Texto inserido

Podemos marcar o texto como um texto inserido para dar ênfase e indicar que ele foi incluído depois. (Existe também a tag <u> para sublinhar, mas ela é uma tag sem significado.)

Texto sobrescrito

Para inseris coisas do tipo x20 usando a tag <sup>

Texto subscrito

Para inserir coisas tipo H2O usando a tag <sub>
