Parágrafos e quebras de linhas

Você pode escrever um texto de qualquer jeito. Basta colocar tudo entre o par de tags <p> e </p>

Se precisar quebrar uma linha use a tag <br>

Vamos usar alguns símbolos especiais:

® = marca registrada
© = copyright
™ = marca registrada em inglês
€ = euro
£ = libra
¥ = iene
¢ = cents
δ = delta
Δ = Delta

Vamos adicionar alguns emojis:

✨ = & + # + x + número héxadecimal referente ao emoji + ;
